기존 REST API가 가지고 있는 문제 해결

- Over-Fetching
  - 불필요한 데이터까지 받는 것
- Under-Fetching
  - 충분한 데이터를 받지 못함
  - 데이터를 얻기 위해 N개의 API를 요청하는 경우
